# Third Party Notices

This repository currently keeps gameplay-critical fauna, ambience, particles and environment feedback procedural unless a local resource pack/soundpack is explicitly provided by the user.

New committed assets must be original, CC0, CC-BY-compatible, or otherwise explicitly permitted for redistribution. Add the asset name, author, source URL, license, and local path here before committing it.

## Procedural Fallbacks

- Wildlife visuals: original multi-part procedural animal models generated by `src/living/EntityModelLoader.ts` and rendered by `src/living/EntityAssetManager.ts`; no external model or texture files.
- Wildlife audio: original runtime-generated spatial one-shots in `src/assets/SpatialAudioMixer.ts` and `src/living/WildlifeAudioSystem.ts`; no external samples.
- Biome ambience: generated WebAudio layers in `src/living/AmbientBiomeAudioSystem.ts`.
- Environment state, fog banks and seasonal color response: generated by `src/environment/*`.

## Adapted Techniques — Volumetric Clouds (no source code copied)

The stratiform cloud renderer (`src/render/weather/StratiformCloudRenderer.ts`),
the fair-weather cumulus field (`src/clouds/FairWeatherCumulusField.ts` +
`src/render/weather/CumulusFieldRenderer.ts`) and their shared pre-baked 3D noise
(`src/render/weather/StratiformNoiseTextures.ts`) are clean-room TypeScript + GLSL
(WebGL2 / Three.js) re-implementations. No upstream
source file, shader, texture, or asset was copied. Techniques (raymarching,
ray-box intersection, shape+detail density, Worley/Perlin-Worley erosion,
Beer-Lambert transmittance, Henyey-Greenstein phase, jitter, early-exit, 3D
noise caching, browser quality/FPS trade-offs) were adapted from:

- **SebLague/Clouds** — MIT, Copyright (c) 2019 Sebastian Lague.
  License: `LICENSES/seb-lague-clouds-MIT.txt`.
- **frmlinn/clouds-sim** — MIT, Copyright (c) 2026 frmlinn.
  License: `LICENSES/clouds-sim-MIT.txt`.
- **mhr1235/cl0ud** — visual inspiration ONLY; no code/shader/asset imported
  (no confirmed license).

Full breakdown of what was adapted vs. only studied: `docs/CLOUDS_REFERENCES.md`.
