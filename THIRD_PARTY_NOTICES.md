# Third Party Notices

This repository currently keeps gameplay-critical fauna, ambience, particles and environment feedback procedural unless a local resource pack/soundpack is explicitly provided by the user.

New committed assets must be original, CC0, CC-BY-compatible, or otherwise explicitly permitted for redistribution. Add the asset name, author, source URL, license, and local path here before committing it.

## Procedural Fallbacks

- Wildlife visuals: original multi-part procedural animal models generated by `src/living/EntityModelLoader.ts` and rendered by `src/living/EntityAssetManager.ts`; no external model or texture files.
- Wildlife audio: original runtime-generated spatial one-shots in `src/assets/SpatialAudioMixer.ts` and `src/living/WildlifeAudioSystem.ts`; no external samples.
- Biome ambience: generated WebAudio layers in `src/living/AmbientBiomeAudioSystem.ts`.
- Environment state, fog banks and seasonal color response: generated by `src/environment/*`.
